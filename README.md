# SharePoint-BirthdayList
SharePoint webpart that lists this month's staff birthdays. Today's birthdays are highlighted, and past birthdays are push down and greyed out.

## BirthdayList
Shows this month's birthdays using manually inserted name, department and picture.

## BirthdayListNew
Shows this month's birthdays using a Person column + Microsoft Graph.
